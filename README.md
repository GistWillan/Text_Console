# Text Console

文字加密解密程序，支持 Base64 和 URL(UTF8) 两种加密算法。

---

## 功能

- 支持 Base64 加密和解密
- 支持 URL(UTF8) 加密和解密
- 记录加密解密历史

---

## 使用方法

1. 运行程序，按照菜单提示选择相应操作。
2. 选择加密或解密算法，输入文本。
3. 查看加密或解密结果。
4. 查看历史记录。

---

- 加密示例：

```bash

2. 开始解密
3. 历史记录
4. 退出程序
